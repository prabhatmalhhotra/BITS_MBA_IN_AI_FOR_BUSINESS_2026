## Sessions 3-4 — Journal, Ledger, Trial Balance, Financial Statements

### Problem 3.1 — Recording transactions in the journal
On 1 April 2025, ABC Traders had the following transactions:
1. Apr 1 — Started business with ₹5,00,000 cash and ₹3,00,000 in bank.
2. Apr 3 — Purchased goods for cash ₹1,20,000.
3. Apr 5 — Purchased goods on credit from XYZ Suppliers ₹80,000.
4. Apr 8 — Sold goods for cash ₹90,000 (cost ₹60,000).
5. Apr 12 — Sold goods on credit to PQR Ltd ₹1,20,000 (cost ₹70,000).
6. Apr 15 — Paid salary ₹25,000 by cheque.
7. Apr 20 — Paid XYZ Suppliers ₹50,000 by cheque.
8. Apr 25 — Received ₹70,000 from PQR Ltd by cheque.

Pass journal entries.

**Solution**

| Date | Particulars | Debit (₹) | Credit (₹) |
|:-----|:------------|----------:|-----------:|
| Apr 1 | Cash A/c   Dr | 5,00,000 |  |
|       | Bank A/c   Dr | 3,00,000 |  |
|       |   To Capital A/c | | 8,00,000 |
|       | *(Capital introduced)* | | |
| Apr 3 | Purchases A/c   Dr | 1,20,000 | |
|       |   To Cash A/c | | 1,20,000 |
| Apr 5 | Purchases A/c   Dr | 80,000 | |
|       |   To XYZ Suppliers A/c | | 80,000 |
| Apr 8 | Cash A/c   Dr | 90,000 | |
|       |   To Sales A/c | | 90,000 |
| Apr 12 | PQR Ltd A/c   Dr | 1,20,000 | |
|       |   To Sales A/c | | 1,20,000 |
| Apr 15 | Salary A/c   Dr | 25,000 | |
|       |   To Bank A/c | | 25,000 |
| Apr 20 | XYZ Suppliers A/c   Dr | 50,000 | |
|       |   To Bank A/c | | 50,000 |
| Apr 25 | Bank A/c   Dr | 70,000 | |
|       |   To PQR Ltd A/c | | 70,000 |

> Note: COGS is NOT recorded in periodic inventory; in perpetual inventory you'd also debit COGS and credit Inventory at cost. Indian convention typically uses periodic — adjust at year-end.

**Common mistakes**: confusing personal vs nominal account rules ("debit the receiver, credit the giver"); reversing Dr/Cr; mixing periodic vs perpetual treatment.

## Sessions 6-7 — Cash Flow Statement (Indirect Method)

### Problem 6.1 — Cash flow from operating activities
ABC Ltd reports for year ended 31-Mar-26:
- Net profit before tax: ₹4,50,000
- Depreciation: ₹85,000
- Loss on sale of equipment: ₹12,000
- Interest expense: ₹35,000
- Trade receivables: increase by ₹70,000
- Inventory: decrease by ₹40,000
- Trade payables: increase by ₹55,000
- Income tax paid: ₹1,20,000

Compute Cash Flow from Operating Activities (CFO).

**Solution (indirect method)**
- Net profit before tax: 4,50,000
- + Depreciation (non-cash): 85,000
- + Loss on sale (non-operating): 12,000
- + Interest expense (financing item — add back): 35,000
- **Operating profit before working capital changes** = 5,82,000
- − Increase in receivables: (70,000)
- + Decrease in inventory: 40,000
- + Increase in payables: 55,000
- **Cash generated from operations** = 6,07,000
- − Income tax paid: (1,20,000)
- **CFO = ₹4,87,000**

> Interest paid would be deducted under financing activities (not operating). Some firms classify interest paid in operating — disclose policy.

## Sessions 8-10 — Ratio Analysis

### Problem 8.1 — Liquidity, Solvency, Profitability ratios
Selected financials of Beta Ltd (₹ lakh):

| Item | 2025 | 2026 |
|------|----:|----:|
| Current assets | 480 | 620 |
| Inventory | 180 | 220 |
| Current liabilities | 200 | 280 |
| Total debt (long-term) | 350 | 400 |
| Total equity | 600 | 700 |
| Net sales | 1,200 | 1,500 |
| COGS | 720 | 900 |
| Operating profit (EBIT) | 240 | 320 |
| Interest | 40 | 50 |
| Tax | 60 | 80 |
| PAT | 140 | 190 |
| Total assets | 1,150 | 1,380 |

Compute and comment: Current Ratio, Quick Ratio, Debt-to-Equity, Interest Coverage, Gross Margin, Net Margin, ROCE, ROE.

**Solution**

| Ratio | Formula | 2025 | 2026 | Comment |
|-------|---------|-----:|-----:|---------|
| Current ratio | CA / CL | 480/200 = **2.40** | 620/280 = **2.21** | >1.5 healthy; slight decline |
| Quick ratio | (CA − Inv)/CL | 300/200 = **1.50** | 400/280 = **1.43** | Adequate liquidity |
| Debt-to-equity | LT debt / Equity | 350/600 = **0.58** | 400/700 = **0.57** | Stable, conservative gearing |
| Interest coverage | EBIT / Interest | 240/40 = **6.0** | 320/50 = **6.4** | Strong ability to service debt |
| Gross margin | (Sales − COGS)/Sales | 480/1200 = **40 %** | 600/1500 = **40 %** | Stable cost structure |
| Net (PAT) margin | PAT/Sales | 140/1200 = **11.7 %** | 190/1500 = **12.7 %** | Improving |
| ROCE | EBIT/(Equity + LT Debt) | 240/950 = **25.3 %** | 320/1100 = **29.1 %** | Better asset productivity |
| ROE | PAT/Equity | 140/600 = **23.3 %** | 190/700 = **27.1 %** | Strong shareholder returns |

**Overall**: Beta is more profitable in 2026 with stable gearing; liquidity slightly weaker but still safe.

### Problem 8.2 — DuPont 3-step decomposition
Continuing Beta Ltd, decompose ROE 2026 via DuPont 3-step.

**Solution**
- **Net margin** = PAT/Sales = 190/1500 = **0.1267 (12.67 %)**
- **Asset turnover** = Sales/Total assets = 1500/1380 = **1.087**
- **Equity multiplier** = Total assets/Equity = 1380/700 = **1.971**
- **DuPont ROE** = 0.1267 × 1.087 × 1.971 = **0.2715 ≈ 27.1 %** ✓ (matches direct computation)

**Insight**: ROE driven primarily by leverage (1.97×) and net margin; turnover modest. To improve further, focus on margins or asset efficiency without over-leveraging.

### Problem 8.3 — Efficiency ratios
For Beta 2026: Avg debtors = 200, Avg inventory = 200, Avg payables = 130.
Compute Receivables days, Inventory days, Payables days, Cash conversion cycle (CCC).

**Solution**
- **Receivables days** = (Avg debtors / Sales) × 365 = (200/1500) × 365 = **48.7 days**
- **Inventory days** = (Avg inventory / COGS) × 365 = (200/900) × 365 = **81.1 days**
- **Payables days** = (Avg payables / COGS) × 365 = (130/900) × 365 = **52.7 days**
- **CCC = Inv days + Recv days − Pay days = 81.1 + 48.7 − 52.7 = 77.1 days**

**Interpretation**: Beta locks cash in working capital for ~77 days. Reducing CCC (faster collection or stretched payables) would free cash.

## Sessions 11-13 — Empirical / Case-style FS Analysis

### Problem 11.1 — Quality of earnings check
Two firms, Alpha Ltd and Gamma Ltd, both report ₹100 cr PAT. But:
- Alpha: CFO = ₹140 cr, capex = ₹50 cr, accruals (∆WC) = +₹10 cr.
- Gamma: CFO = ₹40 cr, capex = ₹130 cr, accruals = +₹70 cr.

Comment on quality of earnings.

**Solution**
- **Alpha**: CFO/PAT = 140/100 = **1.40** (>1, healthy). Free cash flow = 140 − 50 = ₹90 cr (positive). Accruals modest. Earnings backed by cash — high quality.
- **Gamma**: CFO/PAT = 40/100 = **0.40** (cash << reported profit). FCF = 40 − 130 = **−₹90 cr** (negative). Large accruals (+70) suggest aggressive revenue recognition or working capital build-up. **Red flag** — earnings may be inflated; cash needed externally.
- **Conclusion**: Same headline PAT, but Alpha is fundamentally stronger.

### Problem 15.1 — CVP analysis (BEP, MOS)
Standard product: Selling price ₹500/unit, variable cost ₹300/unit, fixed costs ₹16,00,000/year. Current sales = 12,000 units.

(a) Contribution per unit, P/V ratio.
(b) BEP in units and ₹.
(c) Margin of safety in units, ₹, and as % of sales.
(d) Sales required for target profit ₹6,00,000.
(e) New BEP if SP cut by 10 % to clear stock.

**Solution**
- (a) Contribution = 500 − 300 = **₹200/unit**. P/V ratio = 200/500 = **40 %**.
- (b) BEP units = FC / Contribution = 16,00,000 / 200 = **8,000 units**. BEP ₹ = 8,000 × 500 = **₹40,00,000** (or FC/PV = 16,00,000/0.4 = ₹40,00,000).
- (c) MOS units = 12,000 − 8,000 = **4,000 units**; MOS ₹ = 4,000 × 500 = **₹20,00,000**; MOS % = 4,000/12,000 = **33.3 %**.
- (d) Required sales (units) = (FC + Target Profit) / Contribution = (16,00,000 + 6,00,000)/200 = **11,000 units**. Sales ₹ = 11,000 × 500 = ₹55,00,000.
- (e) New SP = 450; new contribution = 450 − 300 = 150. New BEP units = 16,00,000 / 150 ≈ **10,667 units**. *Higher BEP* → more risk; should evaluate price cut carefully.

**Common mistakes**: confusing fixed cost recovery (BEP) with target profit calc; computing P/V ratio off cost instead of selling price.

### Problem 15.2 — Make-or-buy decision
Component X — annual requirement 20,000 units. Make: variable cost ₹120/unit, fixed cost (specifically allocable, avoidable) ₹3,00,000. Buy: vendor quote ₹140/unit. Should the company make or buy? Show breakeven volume.

**Solution**
- **Make total cost** = 20,000(120) + 3,00,000 = 24,00,000 + 3,00,000 = ₹27,00,000.
- **Buy total cost** = 20,000(140) = ₹28,00,000.
- **Make is cheaper by ₹1,00,000**.
- Breakeven volume Q*: 120Q + 3,00,000 = 140Q → 20Q = 3,00,000 → **Q* = 15,000 units**. Above 15,000 units, make is cheaper.

**Other factors to consider**: capacity constraints, vendor reliability, IP/quality, opportunity cost of in-house resources.

### Problem 15.3 — Special order acceptance
Plant operates at 70 % capacity; max capacity 50,000 units. Currently sells 35,000 units at ₹400. Variable cost ₹240/unit; fixed cost ₹40,00,000. New customer offers 10,000 units at **₹290/unit**. Should the order be accepted (assume no impact on regular market)?

**Solution**
- Spare capacity = 50,000 − 35,000 = 15,000 units (enough for 10,000 unit order).
- Incremental revenue = 10,000 × 290 = **29,00,000**.
- Incremental variable cost = 10,000 × 240 = 24,00,000.
- Incremental fixed cost = 0 (sunk + spare capacity).
- **Incremental profit = 29,00,000 − 24,00,000 = ₹5,00,000**.
- Order is at price > variable cost (290 > 240). **Accept** the order.

> If order would crowd out regular sales, the analysis must include opportunity cost of foregone sales.

### Problem 16.2 — Materials variance analysis
Standard: 4 kg/unit @ ₹50/kg. Actual: 4,200 kg used to make 1,000 units, paid ₹2,18,400.

**Solution**
- Standard quantity (SQ) for actual output = 4 × 1,000 = 4,000 kg.
- Standard price (SP) = ₹50/kg. Standard cost of standard quantity = 4,000 × 50 = ₹2,00,000.
- Actual price (AP) = 2,18,400 / 4,200 = ₹52/kg. Actual cost = ₹2,18,400.
- **Material Cost Variance (MCV)** = SQ × SP − AQ × AP = 2,00,000 − 2,18,400 = **₹18,400 (Adverse)**
- **Material Price Variance (MPV)** = (SP − AP) × AQ = (50 − 52) × 4,200 = **₹8,400 (A)**
- **Material Usage Variance (MUV)** = (SQ − AQ) × SP = (4,000 − 4,200) × 50 = **₹10,000 (A)**
- Check: MPV + MUV = 8,400 + 10,000 = 18,400 ✓ = MCV

**Interpretation**: Both higher input price (8,400 A) and higher consumption per unit (10,000 A) hurt cost. Investigate procurement and floor wastage.

## EC-2 / EC-3 quick exam map

| Topic | Likely problem in EC-2 (closed-book) | Likely in EC-3 (open-book) |
|-------|--------------------------------------|----------------------------|
| Journal/Ledger/TB (S3-4) | Pass 6-8 entries + prepare TB; tally check | Combined journal + ledger + adjusted TB |
| IFRS / IAS (S5) | Conceptual: distinguish IFRS from US GAAP on R&D; identify red flags | Apply IFRS 15 (revenue) or IAS 16 (PPE) to a scenario |
| Financial statements (S6-7) | Prepare Trading + P&L + B/S with 4-6 adjustments | Add cash flow statement (indirect) |
| Ratio analysis (S8-10) | Compute 6-8 ratios; one industry classification question | DuPont decomposition + interpret 3-year trend + comparative analysis |
| Case FS analysis (S11-13) | One mini-case; quality of earnings; identify 3 red flags | Full case with multi-year comparative and recommendation |
| Cost accounting + CVP (S14-15) | Cost sheet; BEP & MOS; one decision (make-or-buy or special order) | Multi-product CVP; sensitivity analysis |
| Budgeting + Variance (S16) | Cash budget OR materials variance | Master budget OR full variance set (mat, lab, OH) |

> **Practice tactic**: do the cost-sheet, ratios, and CVP calculations in your head where possible. EC-2 is 2 hrs ~50-60 marks (≈2 min/mark); EC-3 is 2.5 hrs ~80-100 marks. Show every formula and intermediate step — partial credit is ALWAYS available for accounting numericals.
