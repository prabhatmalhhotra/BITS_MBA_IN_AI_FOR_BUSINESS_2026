3 Formula Quick Reference Beginner

Critical Values (Memorise for Mid-Sem)

z-critical: 1.645 (90%), 1.96 (95%), 2.576 (99%)

Chebyshev: ≥ 75% within 2σ; ≥ 88.9% within 3σ

Empirical: 68% / 95% / 99.7% within 1σ / 2σ / 3σ

Core Formulas

Descriptive:
S² = Σ(Xi - X̄)² / (n-1)  |  CV = (S/X̄) x 100%
r = Cov(X,Y) / (Sx Sy)

Probability:
P(A|B) = P(A∩B) / P(B)
Bayes: P(A|B) = P(B|A)P(A) / ΣP(B|Aj)P(Aj)

Distributions:
Binomial: P(X=x) = C(n,x) p^x (1-p)^(n-x); μ=np
Poisson: P(X=x) = e^(-λ) λ^x / x!; μ=λ
Normal: Z = (X-μ)/σ

Inference:
SE(mean) = S/√n  |  SE(prop) = √[p(1-p)/n]
CI(mean): X̄ ± t × S/√n
CI(prop): p̂ ± z × √[p̂(1-p̂)/n]
t-test: t = (X̄ - μ0) / (S/√n)

ANOVA: F = MSB/MSW
Chi-square: χ² = Σ(O-E)²/E
Regression: R² = SSR/SST; b1 = SSXY/SSX
Spearman: rs = 1 - 6Σd² / [n(n²-1)]

5 Open-Book Exam Strategy Beginner

DO

  • Tab your textbook by chapter (use sticky notes)
  • Flag pages with: Z-table, t-table, F-table, χ²-table
  • Bring Casio FX-991EX (non-programmable)
  • Read the entire question before starting
  • Show all steps — partial credit is real
  • State H₀/H₁ explicitly in hypothesis questions

DO NOT

  • Bring handwritten notes or printouts
  • Bring electronic devices (laptop, phone)
  • Waste time looking up basic formulas (memorise the key ones)
  • Skip stating assumptions
  • Forget to write a business conclusion after hypothesis tests
  • Use the wrong test (check the selector table!)
Power Formula Direction (memorise): ↑ sample size (n), ↑ effect size, ↑ α → all INCREASE power (1 - β). More data, bigger effects, and looser thresholds make it easier to detect real differences.
