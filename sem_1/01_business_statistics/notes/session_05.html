Discrete Probability Distributions

When outcomes are countable — defects per batch, conversions per campaign, calls per hour — discrete distributions give you exact probabilities.

Real-World Problem

Reliance Jio: How many customer complaints will the call centre get per hour?

Jio's call centre in Navi Mumbai handles thousands of calls daily. Management needs to staff correctly: too few agents means long wait times, too many means wasted salary. They need to answer: "What is the probability of getting more than 50 calls in the next hour?"

  • Calls per hour is a countable (discrete) variable
  • Calls arrive randomly and independently at an average rate — classic Poisson setup
  • For each call, the caller either gets resolution (yes/no) — could be modelled as Binomial
This session teaches you two workhorse distributions — Binomial (fixed trials, yes/no) and Poisson (rate of rare events) — that model the majority of discrete business scenarios.

1 Discrete Random Variables Beginner

A Discrete Random Variable X takes on a countable number of distinct values (0, 1, 2, 3, ...).

Probability Mass Function (PMF)

The PMF gives P(X = x) for each possible value x. Two rules:

1.   P(X = x) ≥ 0   for all x
2.   Σ P(X = x) = 1   (probabilities sum to 1)

Expected Value (Mean)

E(X) = μ = Σ x · P(X = x)

The long-run average if you repeat the experiment many times. Think of it as the "centre of gravity" of the distribution.

Variance and Standard Deviation

Var(X) = σ² = Σ (x - μ)² · P(X = x)

SD(X) = σ = √Var(X)
Daily Life Example

Suppose you roll a fair die. X = number shown. E(X) = (1+2+3+4+5+6)/6 = 3.5. You can never actually roll 3.5, but on average over hundreds of rolls, your mean will approach 3.5. That is expected value.

2 Binomial Distribution Intermediate

The Setup: BINS Conditions

B
Binary outcomes (success/failure)
+
I
Independent trials
+
N
Fixed Number of trials
+
S
Same probability p each trial

Formula

P(X = x) = C(n, x) · pˣ · (1 - p)ⁿ⁻ˣ

where C(n, x) = n! / [x!(n-x)!]

Mean and Variance

Mean: μ = n · p
Variance: σ² = n · p · (1 - p)
Std Dev: σ = √[n · p · (1 - p)]

Worked Example: Email Marketing

You send n = 500 marketing emails. Historical click-through rate p = 2% (0.02).

Daily Life Example

Cricket analogy: A batsman's strike rate against spin is 40% (p = 0.4). In an innings with n = 20 balls against spin, what is the expected number of scoring shots? μ = 20 × 0.4 = 8 scoring shots. Each ball is a "trial" with two outcomes: score or dot.

Exam Tip: The exam will give you a scenario and expect you to (1) verify it is Binomial by checking BINS conditions, (2) identify n and p, (3) compute the answer. Always state the BINS check explicitly for full marks.

3 Poisson Distribution Intermediate

When to Use Poisson

Models the count of rare events occurring in a fixed interval of time, space, or volume.

P(X = x) = e⁻λ · λˣ / x!

where λ = average rate of occurrence

Key Property

Mean = λ     Variance = λ     (Mean equals Variance!)

Conditions for Poisson

  • Events occur independently
  • Events occur at a constant average rate
  • Two events cannot occur at exactly the same instant
  • The probability of an event in a small interval is proportional to the interval length

Binomial to Poisson Approximation

When n is large (≥ 20) and p is small (≤ 0.05):
Binomial(n, p) ≈ Poisson(λ = np)

Binomial

  • Fixed number of trials (n)
  • Two outcomes per trial
  • Constant probability p
  • Example: 20 items inspected, defective or not
  • Mean ≠ Variance (unless p very small)

Poisson

  • No fixed number of trials
  • Count of events in an interval
  • Average rate λ
  • Example: Calls per hour, defects per sq metre
  • Mean = Variance = λ
Trending Example (2024)

IRCTC website crashes during Tatkal booking: The number of server errors per minute during 10 AM Tatkal rush follows a Poisson distribution. If λ = 3 errors/minute on average, the probability of zero errors in a given minute is P(X=0) = e⁻³ = 0.0498 or about 5%. This is why IRCTC invested in server scaling — the Poisson model predicted that error-free minutes were rare during peak loads.

Daily Life Example

WhatsApp messages: If you receive on average 8 messages per hour (λ = 8), what is the probability of receiving exactly 5 messages in the next hour? P(X=5) = e⁻⁸ × 8⁵ / 5! = 0.0916 or about 9.2%. Poisson works here because messages arrive randomly and independently.

Exam Tip: If the problem says "on average X events per [time/space]," think Poisson. If it says "n trials, each with probability p," think Binomial. Also, if asked "can you use Poisson to approximate this Binomial?" check n ≥ 20 and p ≤ 0.05.

5 Choosing the Right Distribution

Is the variable discrete (countable)?
If yes, continue below. If continuous, go to Session 6.
▼
Is there a fixed number of trials (n)?
Yes = Binomial candidate. No = Poisson candidate.
▼
If Binomial: are BINS conditions met?
Binary, Independent, fixed N, Same p?
▼
If Poisson: events per interval at constant rate?
Independent, constant rate, no simultaneous events?
▼
Apply the appropriate formula!
Remember: Binomial ≈ Poisson when n large, p small, λ = np

Session 5 — Key Takeaways

  • PMF gives P(X = x); all probabilities must sum to 1
  • E(X) = Σ x · P(x) = long-run average; Var(X) = Σ (x - μ)² · P(x)
  • Binomial: BINS conditions, P(X=x) = C(n,x) p^x (1-p)^(n-x), Mean = np, Var = np(1-p)
  • Poisson: rare events per interval, P(X=x) = e^(-λ) λ^x / x!, Mean = Var = λ
  • Binomial → Poisson approximation when n ≥ 20 and p ≤ 0.05
  • Excel: BINOM.DIST and POISSON.DIST; FALSE = PMF, TRUE = CDF
