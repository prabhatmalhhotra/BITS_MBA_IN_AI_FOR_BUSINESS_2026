# MPO — One-Page Revision Sheet

## OB pyramid
Individual → Group → Organisation. Disciplines: Psychology, Sociology, Social Psych, Anthropology, Political Science.

## Personality
**Big Five (OCEAN)**: Openness · Conscientiousness · Extraversion · Agreeableness · Neuroticism. Conscientiousness predicts performance most.

## Perception biases
Halo · Stereotyping · Projection · Contrast · Self-serving · Fundamental attribution error.

Attribution: Distinctiveness, Consensus, Consistency.

## Learning
Classical (Pavlov) · Operant (Skinner: reinforcement schedules) · Social (Bandura — observation).

## Motivation theories

| Theory | Key insight |
|--------|------------|
| Maslow | 5-level hierarchy |
| McGregor | X (control) vs Y (empower) |
| Herzberg | Hygiene prevents dissatisfaction; Motivators drive engagement |
| McClelland | nAch · nAff · nPow |
| Vroom (Expectancy) | M = E × I × V |
| Adams (Equity) | Compare input/output ratios |
| Locke (Goal-Setting) | SMART + difficult + feedback |
| Hackman-Oldham (JCM) | MPS = ((SV+TI+TS)/3)×A×F |
| Self-Determination | Autonomy, Competence, Relatedness |

## Groups & teams
**Tuckman**: Forming → Storming → Norming → Performing → Adjourning.
**Groupthink** symptoms: invulnerability, rationalisation, peer pressure, self-censorship, illusion of unanimity.
Decision tools: brainstorming, NGT, Delphi, e-meetings.

## Communication
Sender → Encoding → Channel → Decoding → Receiver → Feedback.
Barriers: filtering, selective perception, overload, emotion, language.
High-context (India, Japan) vs Low-context (US, Germany).

## Leadership

| Theory | Core idea |
|--------|----------|
| Trait | Born leaders' qualities |
| Behavioural (Ohio, Mgr Grid) | Task vs people |
| Fiedler | LPC × situation favourableness |
| Hersey-Blanchard | Style matches follower readiness (S1–S4) |
| Path-Goal (House) | Leader clarifies path |
| Transformational | 4 I's: Idealised influence, Inspirational motivation, Intellectual stimulation, Individualised consideration |
| Transactional | Contingent reward, MBE |
| Authentic | Self-awareness, transparency, ethics |
| Servant | Serve first |

## Power (French & Raven)
Legitimate · Reward · Coercive · Expert · Referent.
Personal (Expert + Referent) > Positional.

## Conflict (Thomas-Kilmann)
Competing · Collaborating · Compromising · Avoiding · Accommodating (axes: assertiveness × cooperativeness).

## Negotiation
Distributive (win-lose) vs Integrative (win-win). BATNA.

## Structure
6 elements: specialisation, departmentalisation, chain of command, span of control, centralisation, formalisation.
Mechanistic (stable) vs Organic (dynamic).

## Culture (Schein)
Artefacts → Espoused values → Underlying assumptions.

## Hofstede dimensions
PDI · IDV · MAS · UAI · LTO · IND.

## Change models

**Lewin**: Unfreeze → Change → Refreeze.

**Kotter (8 steps)**: Urgency → Coalition → Vision → Communicate → Empower → Short-term wins → Sustain → Anchor.

**ADKAR**: Awareness · Desire · Knowledge · Ability · Reinforcement.

Resistance sources (individual): habit, security, fear, selective info. Organisational: structural inertia, threats to power/expertise.

## HRM functions
Acquisition (workforce planning, recruit, select) · Development (T&D) · Motivation (compensation) · Maintenance (retention, ER).

ADDIE: Analysis → Design → Develop → Implement → Evaluate.
Kirkpatrick: Reaction → Learning → Behaviour → Results.

## Stress
Sources: environmental, organisational, personal. Symptoms: physiological, psychological, behavioural. Coping: time mgmt, exercise, social support, EAP, wellness programmes.

## DEI
Diversity = mix; Equity = fairness; Inclusion = mix is heard.
Business case: better decisions, wider talent, better customer view.

## Last-minute case answer template

1. Frame: name of framework being applied.
2. Diagnose root causes (3–4 bullet points).
3. Recommend (3–4 actions, time-boxed).
4. Metrics for success.
5. Risks + mitigation.
