## Manager fundamentals (Session 1)

- **Fayol's functions**: Planning, Organising, Leading, Controlling.
- **Mintzberg's roles** (10 roles, 3 categories): Interpersonal (Figurehead, Leader, Liaison), Informational (Monitor, Disseminator, Spokesperson), Decisional (Entrepreneur, Disturbance Handler, Resource Allocator, Negotiator).
- **Katz's skills**: Technical, Human, Conceptual (conceptual ↑ at higher levels).
- **OB disciplines**: Psychology, Social Psychology, Sociology, Anthropology, Political Science.

## HRM (Session 2)

- **Ulrich's 4 HR roles**: Strategic Partner, Administrative Expert, Employee Champion, Change Agent.
- **HRM functions**: Acquisition, Development, Motivation, Maintenance.
- **PM vs HRM**: Transactional/Compliance ↔ Strategic/Value-creation.

## Individual (Sessions 3-8)

### Attitudes & Diversity
- **Components of attitude**: Cognitive (belief), Affective (feeling), Behavioural (intention).
- **Cognitive dissonance** → discomfort from inconsistency, motivates change.
- **OCB**: Discretionary helpful behaviour beyond formal job.
- **Diversity**: Surface-level (demographic) vs Deep-level (values, personality).

### Personality
- **Big Five (OCEAN)**: Openness, Conscientiousness (best predictor), Extraversion, Agreeableness, Neuroticism.
- **MBTI**: 16 types, 4 dichotomies (E/I, S/N, T/F, J/P) — popular but limited.
- **Hofstede dimensions**: Power Distance, Individualism-Collectivism, Masculinity-Femininity, Uncertainty Avoidance, LT Orientation, Indulgence.

### Perception & Decisions
- **Fundamental Attribution Error**: Over-weight personal factors when judging others.
- **Self-serving bias**: Own success = internal; failure = external.
- **Bounded rationality (Simon)**: Satisficing, not optimising.
- **Common biases**: Anchoring, Availability, Confirmation, Overconfidence, Hindsight, Escalation.

### Emotions
- **Emotion vs Mood**: Intense+brief+specific cause vs mild+longer+vague cause.
- **Emotional Labour**: Surface acting (fake) vs Deep acting (modify inner) — deep healthier.
- **Goleman EI 4 components**: Self-awareness, Self-management, Social-awareness, Relationship-management.

### Motivation theories

| Theory | Core idea | Key takeaway |
|--------|-----------|--------------|
| **Maslow** | 5-level need hierarchy | Lower needs first |
| **McGregor X/Y** | Workers dislike vs enjoy work | Style depends on assumption |
| **Herzberg** | Hygiene ≠ Motivators | Fix hygiene, then motivate |
| **McClelland** | nAch / nAff / nPow | Profile for fit |
| **Equity (Adams)** | Compare I/O ratio with peers | Fairness matters |
| **Expectancy (Vroom)** | M = E × I × V | All three must hold |
| **Self-Determination** | Intrinsic via Autonomy/Competence/Relatedness | Beware extrinsic crowding |
| **Goal-Setting (Locke)** | SMART goals | Specific + difficult > "do best" |
| **Reinforcement (Skinner)** | Behaviour driven by consequences | Positive reinforcement strongest |

### Motivation applications
- **JCM**: SV, TI, TS, Autonomy, Feedback → MPS = ((SV+TI+TS)/3) × A × F.
- **Job design**: Enrichment (vertical), Enlargement (horizontal), Rotation, Flex-time, Telework.
- **MBO**: Joint goal-setting, periodic review.
- **OKR**: Objectives + Key Results, quarterly cadence.
- **Variable pay**: Piece-rate, Bonus, Profit-sharing, Gain-sharing, ESOP.

## Group (Sessions 9-10)

- **Group vs Team**: Sum of contributions vs collective effort with shared accountability.
- **Tuckman**: Forming → Storming → Norming → Performing → Adjourning.
- **Cohesiveness × Norms**: high+aligned → high productivity.
- **Risks**: Groupthink (override realism), Social loafing (less effort), Polarisation (extreme decisions).
- **Team types**: Problem-solving, Self-managed, Cross-functional, Virtual.
- **Communication directions**: Downward, Upward, Lateral, Diagonal.
- **Barriers**: Filtering, Selective perception, Information overload, Emotions, Language, Silence.

## Leadership (Session 11)

| Theory | Core |
|--------|------|
| Trait | Leaders born with traits (modest validity); Big Five + EI |
| Behavioural (Ohio State) | Initiating Structure + Consideration |
| Fiedler Contingency | LPC × Situation favourableness |
| Hersey-Blanchard | Telling → Selling → Participating → Delegating (by readiness) |
| Path-Goal (House) | Clear path; remove obstacles; reward |
| Transactional | Exchange; rewards-for-performance |
| Transformational | Vision, Intellectual stimulation, Inspirational motivation, Individualised consideration, Idealised influence |
| Servant | Serve followers; growth focus |
| Authentic | Self-aware; values-based; transparent |

EI critical at senior levels (meta-analytic evidence).

## Organisation (Sessions 12-13)

- **6 elements of structure**: Specialisation, Departmentalisation, Chain of command, Span of control, Centralisation, Formalisation.
- **Mechanistic** (rigid, stable env) vs **Organic** (flexible, dynamic env).
- **Structures**: Functional, Divisional, Matrix, Network/Boundaryless.
- **Schein's 3 levels of culture**: Artifacts → Espoused values → Underlying assumptions.
- **Strong vs weak culture**: Intensity + sharing.
- **Lewin**: Unfreeze → Change → Refreeze.
- **Kotter 8 steps**: Urgency → Coalition → Vision → Communicate → Empower → Wins → Sustain → Anchor.
- **Force field analysis**: Drivers vs Restrainers.

## Recruitment & Selection (Session 14)

- **Sources** — Internal: promotion, transfer, referral. External: ads, campus, agencies, online, social.
- **Selection steps**: Application → Screening → Tests → Interview → Reference → Medical → Offer.
- **Test types**: Ability/Aptitude, Personality, Interest, Achievement, Work sample, Assessment centre.
- **Interview**: Structured > Unstructured (validity).
- **RJP** (Realistic Job Preview) → reduces early turnover.
- **Selection errors**: Halo, Similar-to-me, Contrast, Primacy/Recency, Stereotyping.

## Compensation (Session 15)

- **Components**: Direct (base, allowances, incentives, bonus); Indirect (PF, gratuity, insurance, leave, perks).
- **Job evaluation methods**: Ranking, Job Classification, Point method, Factor comparison.
- **Internal equity** (job evaluation) vs **External equity** (market survey).
- **Pay-for-performance**: Merit pay, bonus, ESOPs, profit-share, gain-sharing.
- **Broadbanding**: Few wide bands → flexibility.
- **Total rewards**: Comp + Benefits + Work-life + Recognition + Development.

## Performance Appraisal & Management (Session 16)

| Method | Notes |
|--------|-------|
| Ranking | Order all employees |
| Forced Distribution (Bell-curve) | Fixed % per grade — controversial |
| Graphic Rating Scale | Trait-based; subjective |
| Critical Incident | Specific examples |
| Essay | Free-form; rater bias |
| BARS | Behavioural anchors; high reliability |
| MBO | Joint goals; outcome-based |
| 360-degree | Multi-source; comprehensive but expensive |
| Assessment Centre | Multi-method; high cost |
| OKR | Objectives + Key Results; quarterly |

- **Errors**: Halo, Horn, Leniency, Strictness, Central tendency, Recency, Similarity, Stereotyping.
- **Performance Management** > Performance Appraisal: continuous, developmental, integrated cycle.
