# Operations Management — Short Questions

> 30 short Q&A. 2–5 marks.

**Q1. Define operations management.** Set of activities that create value in goods/services by transforming inputs (labour, capital, material, info) to outputs.

**Q2. Differentiate productivity and efficiency.** Productivity = Output / Input (broad); Efficiency = Actual / Effective capacity (narrower, capacity-utilisation focus).

**Q3. List 10 strategic OM decisions.** Goods/Services design, Quality, Process & capacity, Location, Layout, HR/Job design, Supply chain, Inventory, Scheduling, Maintenance.

**Q4. State 4 competitive priorities.** Cost, Quality, Time (response/delivery), Flexibility.

**Q5. Compute single-factor labour productivity: 800 units, 40 labour-hours.** = 800/40 = **20 units/hr**.

**Q6. Define critical path.** Longest path through project network; determines minimum project duration; activities on it have zero slack.

**Q7. PERT activity time formula.** t = (a + 4m + b)/6 where a = optimistic, m = most likely, b = pessimistic.

**Q8. Compute t and σ for a = 4, m = 6, b = 14.** t = (4 + 24 + 14)/6 = 7; σ = (14−4)/6 = 1.67; σ² = 2.78.

**Q9. Difference between CPM and PERT.** CPM: deterministic times, focus on cost-time trade-off. PERT: probabilistic times, focus on completion probability.

**Q10. Define MAD.** Mean Absolute Deviation = Σ|error|/n. Measures average forecast error magnitude.

**Q11. Exponential smoothing formula.** F(t+1) = α·A(t) + (1−α)·F(t). α∈[0,1].

**Q12. Define QFD.** Quality Function Deployment — methodology to translate customer needs ("voice of customer") into engineering characteristics.

**Q13. Define TQM.** Total Quality Management — organisation-wide approach focused on continuous improvement, customer satisfaction, employee involvement.

**Q14. Steps in DMAIC.** Define, Measure, Analyse, Improve, Control.

**Q15. List 7 QC tools.** Cause-effect, Pareto, Control chart, Scatter, Histogram, Check sheet, Flow chart.

**Q16. Differentiate Cp and Cpk.** Cp = process potential ((USL−LSL)/6σ); Cpk = process actual capability accounting for centring (min of two one-sided indices).

**Q17. Define common cause vs special cause.** Common = inherent process variation. Special = assignable, identifiable cause (operator change, machine fault).

**Q18. Compute UCL/LCL for p chart: p̄ = 0.05, n = 100.** σ_p = √(0.05×0.95/100) = 0.0218. UCL = 0.05 + 3×0.0218 = **0.116**. LCL = 0.05 − 3×0.0218 = **−0.016 → 0**.

**Q19. Define utilization vs efficiency.** Utilization = Actual / Design capacity. Efficiency = Actual / Effective capacity.

**Q20. EOQ formula.** Q* = √(2DS/H). Annual cost at Q* = √(2DSH).

**Q21. EOQ for D = 5,000, S = ₹100, H = ₹4.** Q* = √(2×5000×100/4) = √250,000 = **500 units**.

**Q22. Define Reorder Point.** Inventory level at which a new order is placed. ROP = d × L + Safety stock.

**Q23. Define Safety Stock for service level.** SS = Z × σ_d × √L. Z from normal distribution for chosen service level.

**Q24. Differentiate chase, level and mixed aggregate strategies.** Chase = adjust workforce/output to demand. Level = constant output, absorb variation in inventory. Mixed = blend.

**Q25. List Johnson's rule steps.** For 2-machine flow shop: find shortest time; if on M1 → schedule first available, if on M2 → schedule last available; cross out and repeat.

**Q26. List 5 sequencing rules.** FCFS, SPT, EDD, LPT, CR (Critical Ratio).

**Q27. Define Theory of Constraints (TOC).** Throughput limited by bottleneck; improve bottleneck for biggest leverage. Steps: Identify, Exploit, Subordinate, Elevate, Repeat.

**Q28. List 8 wastes (TIMWOODS).** Transportation, Inventory, Motion, Waiting, Overproduction, Overprocessing, Defects, Skills (under-utilised people).

**Q29. Define Bullwhip Effect.** Amplification of demand variability upstream in supply chain — small retail variation → large factory swings. Caused by order batching, demand forecasting in series, rationing, price fluctuations.

**Q30. Define MTBF.** Mean Time Between Failures = Total operating time / Number of failures.
